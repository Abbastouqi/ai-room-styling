# Model Storage

Download all models using:
```bash
python scripts/download_models.py
```

## Required Models:
1. **MiDaS** - Depth estimation
2. **YOLOv8** - Object detection
3. **SAM** - Segmentation
4. **Stable Diffusion** - Image generation
5. **ControlNet** - Guided generation

Models are gitignored due to size (~15-20 GB total).
